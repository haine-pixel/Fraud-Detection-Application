About the Fraud Detection Model

A machine learning-based fraud detection system that achieved 99.43% accuracy in identifying fraudulent transactions using XGBoost classification.

Dataset: 10,000 synthetic transactions | Features: Transaction amount, country, transaction type, merchant category | Challenge: Highly imbalanced dataset (5% fraud rate)

How It Works

The fraud detection system analyzes transaction patterns using a comprehensive machine learning approach:

1. Data Analysis

Analyzed 10,000 synthetic transactions with a 5% fraud rate. Conducted exploratory data analysis using correlation heatmaps and feature distribution analysis to identify key fraud indicators.

2. Feature Engineering

Removed low-correlation features (hour, user_id) and applied StandardScaler to transaction amounts. Used One-Hot Encoding for categorical features including country, transaction type, and merchant category.

3. Model Selection

Compared three algorithms: Logistic Regression (98.87% accuracy), Random Forest (99.03% accuracy), and XGBoost (99.43% accuracy). XGBoost was selected for its superior fraud detection recall of 98.67%.

4. Pipeline Deployment

Created a scikit-learn pipeline integrating preprocessing and XGBoost classification. The model handles class imbalance using scale_pos_weight and optimized hyperparameters for real-time prediction.
